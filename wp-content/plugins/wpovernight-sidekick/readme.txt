== Description ==

WP Overnight Sidekick is an administration plugin for all your WP Overnight Purchases. All WP Overnight themes and plugins will be managed via this plugin.

== Installation ==

* Download the plugin zip file
* In WordPress, go to Plugins > Add New, then click the “Upload Plugin” button on the top of the page
* Browse to the zip file, then press ‘Install Now’.
* After the upload/installation is finished, press the ‘Activate Plugin’ link – done!

You can now manage your license keys under WP Overnight > Manage Licenses
